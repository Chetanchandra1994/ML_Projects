Conclusion of the Weight Loss Study¶

  • All the test results indicate towards the same thing i.e the average weight has increased after implementing the diet.

  1. 2 sided test:

    • There was a difference between the before and after weight, but we weren't clear if the average weight had increased or decreased.

  2. Left Tailed Test:

    • Here we failed to reject the null hypothesis, indicating that either the average weight remained the same or had increased, but according to our 2 sided test, we know that the average weight wasn't the same, there was some change.
    • We could've conculded here only that our average weight had increased but to solidify our claim we performed a right tailed as well.

  3. Right Tailed Test:

    • Here we had sufficient evidence to reject the null hypothesis, meaning now we are clear after conducting the 3 tests that the average weight after implementing the diet had increased.

MY UNDERSTANDING¶

So essentially in python the 2 sided, left side and right side methods work similarly.¶

  • For a 2 sided test the code is
    • stats.ttest_1samp or stats.ttest_ind or stats.ttest_rel
    • but they all return a pvalue for both side(area under both ends of the curve) .
  • For a left tailed test the code is
    • stats.ttest_1samp or stats.ttest_ind or stats.ttest_rel (..... alternative='less')
    • but they all return a pvalue for the left side (area under the left side of the curve) .
  • For a right tailed test the code is
    • stats.ttest_1samp or stats.ttest_ind or stats.ttest_rel (..... alternative='greater')
    • but they all return a pvalue for the right side (area under the right side of the curve) .
